SeeCodeRun

SeeCodeRun is a project that started at George Mason University with the goal of being a pastebin that would help developers understand and code better Javascript. Interns were allowed to add features to the project during the Summer Internship, and my contributions are listed below

  • Color coordination on visual elements to code that changes their properties
  • Hover actions on lines of code that affect web elements which reveal elements changed by that line of code
  • Hover action on web elements in the pastebin which highlights affecting code in the corresponding color. Gives developers a way to find important lines of code faster

Photon

Productivity app that integrates with the iOS Calendar to manage events and provide a visually appealing view of events

  • To-Do List from which tasks can be added, removed, and scheduled
  • Timer view which shows the remaining time allocated for a task (event, homework, project, etc.)
  • Future-view which uses arcs to show events for the next three hours. User can drag along arc to see all coinciding tasks at a given time
  • Achieved over 9,000 downloads on Apple App Store
